# 🏪 Optines - Solution de Gestion pour PME avec Activité Physique

## 📋 Présentation Générale

**Optines** est une application mobile complète dédiée aux **PME possédant une activité sur site avec vente de produits physiques sur place**. Conçue spécifiquement pour les magasins agroalimentaires, boulangeries, épiceries, et commerces de proximité, elle optimise la gestion quotidienne de vos équipes et de vos opérations.

## 🔧 **Installation et Démarrage**

### **Démarrage Rapide (5 minutes)**
```bash
# 1. Installation des dépendances
npm install

# 2. Configuration Supabase
npm run setup-supabase

# 3. Démarrage complet
npm start
```

### **Configuration Requise**
- **Serveur Node.js** (port 3001)
- **Base de données Supabase** configurée
- **Compte utilisateur** avec droits administrateur

### **Scripts Disponibles**
```bash
npm start              # Démarrage complet
npm run dev-full       # Logs détaillés
npm run check-health   # Vérification système
npm run restart        # Redémarrage complet
```
